* [Installation](#installation)

## Installation

Install the latest version with

```bash
$ composer require codekandis/types
```

## Testing

Test the code with

```bash
$ composer run-script test
```

If you like to retrieve a full coverage report run

```bash
$ composer run-script test-coverage
```
